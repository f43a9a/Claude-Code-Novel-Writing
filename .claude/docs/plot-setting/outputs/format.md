## 📋 このファイルの役割
`/plot-setting` コマンドが生成するGitHub Issue更新とローカルファイルの標準テンプレートを定義します。
詳細分析と進行管理を統合した、一元的なプロジェクト管理を目的とします。

## 🎯 GitHub Issue・ローカルファイル共通テンプレート
